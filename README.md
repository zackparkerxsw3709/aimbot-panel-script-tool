## About the Panel

AIMBOT Panel is a lightweight HTML web interface organized around game-script style use. Its static page layout brings panel navigation and aimbot-related functionality together in a simple browser-accessible UI.

Because the project is delivered as HTML, it can be deployed through static hosting or modified directly without setting up a larger application framework. It is intended for users who need an easily opened browser panel that can be adjusted manually and used alongside a game-script workflow.
